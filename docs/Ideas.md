# 💡 Rwal Ideas Backlog

> ⚠️ **This is a brainstorming list, not a roadmap.**  
> These ideas may or may not be implemented. If something gains traction, it moves to [Roadmap](https://github.com/Aloncie/Rwal/blob/main/docs/Roadmap.md)

## 🧩 Feature Ideas
- [ ] **Multi‑monitor support** – different wallpapers per screen.
- [ ] **Add purity check** – check for bad wallpapers (e.g., NSFW or pornographic content).
- [ ] **Add `atleast` and `random` options** - make wallpaper selection more flexible.
- [ ] **Add all config options in TUI settings** - give users control over all configuration options.
- [ ] **Add `TIMEOUT` option in settings** - allow users to set a timeout for the wallpaper, e.g. for download large 4k images.

## 🛠 Technical Experiments
- [ ] **Async logging** – non‑blocking log writes.
