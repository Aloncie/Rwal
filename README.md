💦 __Rwal__ - the program to refresh your wallpaper using keywords from any space of internet💫<br><br>

_______

### 💨Installation💨

``` bash
git clone https://github.com/Aloncie/Rwal.git
cd Rwal
mkdir build && cd build
cmake ..
make
sudo make install
```
