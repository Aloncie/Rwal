**Rwal** is a hybrid CLI wallpaper utility that combines an interactive terminal interface with a silent, daemonвҖ‘like mode for regular wallpaper changes. It fetches images from [Wallhaven](https://wallhaven.cc). Built with modern C++20 and Qt5.

---

## вңЁ Features

- **Interactive menu** вҖ“ navigate, add/remove keywords, configure timer.
- **Silent mode** вҖ“ oneвҖ‘shot wallpaper change, ideal for cron/systemd.
- **HotвҖ‘reload config** вҖ“ edit `config.json` while the app runs.
- **Smart keyword handling** вҖ“ random selection, manual editing via `$EDITOR`.
- **Systemd timer integration** вҖ“ set hourly/daily wallpaper rotation.
- **BuiltвҖ‘in logging** вҖ“ with autoвҖ‘rotation and permission handling.
- **Modular design** вҖ“ clear separation of UI, networking, and platform layers.

## рҹҡҖ Installation & Quick Start

```bash

# 2. Use
```
